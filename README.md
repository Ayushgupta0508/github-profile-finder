# JavaScript GitHub Profile Finder 

This is simple Github Profile Finder application build using HTML , CSS , JavaScript. Users can search for Github profiles by entering a username.

## Feature

- Search people by username.
- Dynamic Profile Image
- Name
- Bio
- Repositories
- Followers
- Following
- Location
- GitHub Profile Link
- Loading State
- Error Handling 

## Technology Used

- HTML5
- CSS3
- Javascript (ES6+)
- GitHub API

## What I Learned

- Working with REST APIs
- Using fetch() with async/await
- Handling API and network errors
- Managing Loading States
- Organizing code into reusable function
